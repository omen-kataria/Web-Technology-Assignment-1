# Web Technology Assignment-1

## Personal Portfolio Website

This project is a Personal Portfolio Web Page developed using **HTML5 and CSS3 only** as part of **Web Technology Assignment-1**.

The purpose of this assignment is to design and develop a responsive portfolio website that demonstrates basic front-end development skills such as structured HTML layout, CSS styling, responsive design, and organized content presentation.

JavaScript and external frameworks are not used in this project.

## Website Sections

### Header Section

The header contains:

* Student name
* Profile image
* Navigation menu with links to:

  * Home
  * About
  * Skills
  * Projects
  * Contact

### About Me Section

This section includes:

* A short biography
* Education details displayed using a table

### Skills Section

Displays technical skills with **CSS styled progress bars**.

Example skills:

* Linux Administration
* HTML
* CSS
* Networking
* Cybersecurity Basics

### Projects Section

Includes at least **two sample projects**. Each project contains:

* Project title
* Short description
* Project image

### Contact Section

Contains a contact form with:

* Name field
* Email field
* Message field
* Submit button

### Footer

The footer includes:

* Social media links
* GitHub profile
* LinkedIn profile
* Copyright text

## CSS Features Used

The project uses the following CSS concepts:

* External CSS file (`style.css`)
* CSS selectors (element, class, id)
* Box model (margin, padding, border)
* Flexbox layout
* Hover effects
* Background colors and gradients
* Custom fonts using Google Fonts
* CSS animations
* Responsive design using media queries
